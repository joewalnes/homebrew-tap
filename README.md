# Joewalnes Tap

## How do I install these formulae?

`brew install joewalnes/tap/<formula>`

Or `brew tap joewalnes/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joewalnes/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
